## 一、一句话目标

把「一根两端固定温度的铁棍，求棍上各点的稳态温度」建模成 $Ax=b$，用 **Jacobi** 和 **Gauss-Seidel** 两种迭代法求解，与直接解对比，画三类图，分析收敛性与矩阵性质的影响。

## 二、工程画面（这是什么问题）

```
        左端烧到 100°C                 右端泡冰水 0°C
           ●━━━━━━┯━━━━━━┯━━━━━━┯━━━━━━┯━━━━━━┯━━━━━━●
                 T1     T2     T3     T4     T5
                 ←——— 棍上 5 个内部点的温度，未知 ———→
```

- **物理规律**：稳态下，每个内部点的温度 = 左右两个邻居的平均值。
- **边界**：两端温度已知（100°C 与 0°C）。
- **求解**：中间 n 个点的温度。

这就是任务书要求的「将一个实际或模拟问题表示为 $Ax=b$」。

## 三、数学模型层（评分：模型建立 3 分）

### 3.1 连续问题

一维稳态热传导（无内热源），控制方程为

$$T''(x)=0$$

边界条件 $T(0)=100,\ T(L)=0$。解析解是线性下降：

$$T(x)=100\left(1-\dfrac{x}{L}\right)$$

迭代结果最终应逼近这条直线。

### 3.2 离散化 → $Ax=b$

把棍等分为 $n+1$ 段，内部节点 $i=1..n$。用二阶中心差分：

$$-T_{i-1}+2T_i-T_{i+1}=0$$

代入两端边界（$T_0=100,\ T_{n+1}=0$），5 个内部点拼成：

```
[ 2 -1  0  0  0 ] [T1]   [100]
[-1  2 -1  0  0 ] [T2]   [  0]
[ 0 -1  2 -1  0 ] [T3] = [  0]
[ 0  0 -1  2 -1 ] [T4]   [  0]
[ 0  0  0 -1  2 ] [T5]   [  0]
      矩阵 A         x      向量 b
```

### 3.3 A / x / b 的含义

| 符号 | 含义 | 本例 |
|---|---|---|
| **A** | 棍的结构关系（谁挨着谁） | n×n 三对角矩阵，对角线 2、上下次对角 -1 |
| **x** | 要求的未知数 | 棍上 n 个内部点的温度 $[T_1,\dots,T_n]^T$ |
| **b** | 已知的边界贡献 | $[100, 0, \dots, 0]^T$（两端固定温度搬过来的） |

### 3.4 矩阵性质（决定迭代成败 —— 结果分析的核心）

- **对角占优**：每行 $|对角|=2=|{-1}|+|{-1}|$，是「弱对角占优 + 不可约」→ 保证 Jacobi / GS 收敛。
- **谱半径**（收敛速度的标尺）：
  - Jacobi：$\rho_J=\cos\!\left(\dfrac{\pi}{n+1}\right)$
  - Gauss-Seidel：$\rho_{GS}=\rho_J^{\,2}=\cos^2\!\left(\dfrac{\pi}{n+1}\right)$
  - 两者都 $<1$，故都收敛。
- **规模效应**：$n$ 越大，$\rho$ 越接近 1，收敛越慢——这是分析要点。

## 四、计算实现层（评分：算法实现 4 分）

### 4.1 Jacobi 迭代

$$T_i^{(k+1)}=\frac{b_i+T_{i-1}^{(k)}+T_{i+1}^{(k)}}{2}$$

特点：算第 $k+1$ 轮时，邻居全部用**上一轮的旧值**。

### 4.2 Gauss-Seidel 迭代

$$T_i^{(k+1)}=\frac{b_i+T_{i-1}^{(k+1)}+T_{i+1}^{(k)}}{2}$$

特点：左邻居用**刚算出的新值**，右邻居用旧值；信息传递更快，收敛约快 2 倍。

### 4.3 参数与停止条件

| 项 | 取值 |
|---|---|
| 初值 $x^{(0)}$ | 全 0 向量 |
| 停止条件 | $\lVert x^{(k)}-x^{(k-1)}\rVert_\infty < 10^{-6}$ 或 达到 `max_iter=10000` |
| 直接解参照 | `numpy.linalg.solve(A, b)` |
| 记录 | 每步误差、残差 $\lVert Ax^{(k)}-b\rVert$、耗时 |

## 六、结果分析层（评分：结果分析 4 分）

报告里要讲清四点（对应评分点「收敛性、误差、方法差异、原因」）：

1. **收敛性**：两方法都收敛（$\rho<1$）。
2. **收敛速度**：GS 比 Jacobi 快约 2 倍，对应 $\rho_{GS}=\rho_J^2$。
3. **误差**：与直接解的差 $<10^{-6}$，达到阈值。
4. **矩阵性质的影响**（拿满分的关键）：弱对角占优保证收敛；$n$ 增大 $\rho\to1$，迭代变慢。

**加分项 · 不收敛反例**（强烈建议加）：另造一个**非对角占优**矩阵，例如 $A=\begin{bmatrix}1&2\\2&1\end{bmatrix}$，其 Jacobi 谱半径 $>1$，迭代发散。一张发散的误差曲线图，直接证明「矩阵性质决定迭代效果」——这正好命中评分点「矩阵性质对迭代效果的影响」。
