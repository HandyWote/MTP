## 一、一句话目标

把「估计一批右偏寿命数据的总体中位数 $m$」这个**没有 t/Wilson/Welch 式光滑、密度无关的闭式区间**的估计问题，用 **非参数 Bootstrap（percentile 主、BCa 对照）** 解决，用 Exp(1) 模拟（真中位数 $\ln 2$）做覆盖率可信度自证，画三类图，分析样本量 / 重采样次数 / 数据波动三因素的影响。

## 二、工程画面（这是什么问题）

```
   原始样本 x = (x₁, x₂, …, xₙ)   n=24，波音720第7架机空调故障间隔(小时)
        ┌─────────────────────────────────────┐
        │  3 5 5 13 14 15 22 22 23 30 36 39   │
        │  44 46 50 72 79 88 97 102 139 188   │
        │           197 210                   │
        └──────────────────┬──────────────────┘
                           │ 有放回抽 n 个（同分布、允许重复）
              ┌────────────▼────────────┐
        第 1 次│ 13 22 5 210 14 22 …     │ → 中位数 m̂*₁ = 43
        第 2 次│ 5 5 97 39 188 30 …      │ → 中位数 m̂*₂ = 39
          ⋮    │  ⋮   ⋮   ⋮   ⋮   ⋮      │      ⋮
        第 B 次│ 22 30 72 3 139 97 …     │ → 中位数 m̂*_B = 47
              └────────────┬────────────┘
                           │  B = 10000 个 bootstrap 中位数
                           ▼
              ┌──────────────────────────┐
              │  m̂* 的经验分布 Ĝ*         │ → 点估计 m̂ = median(x)
              │   直方图 + 核密度          │   标准误 SE* = std(m̂*_b)
              └─────────────┬────────────┘
                            │
              percentile CI │  BCa CI（偏差+加速度修正）
              [Q(0.025),    │  [Q(α₁), Q(α₂)]，α₁/α₂ 由 z₀、â 调整
               Q(0.975)]    │
```

- **重采样**：从原始样本**有放回**抽 $n$ 个，得到一个 bootstrap 样本 $x^{*b}$。
- **统计量**：对每个 $x^{*b}$ 算中位数 $\hat{m}^{*b}$，重复 $B=10000$ 次。
- **分布**：这 $B$ 个 $\hat{m}^{*b}$ 的经验分布 $\hat{G}^*$ 近似 $\hat{m}$ 的真实抽样分布—— percentile CI 直接取 $\hat{G}^*$ 的分位数。

这就是任务书要求的「从原始样本中进行有放回重采样，重复多次生成 Bootstrap 样本……形成 Bootstrap 估计分布」。

## 三、数学模型层（评分：问题与数据 3 分）

### 3.1 估计目标与统计量

| 项 | 定义 | 本例取值 |
|---|---|---|
| 总体中位数 $m$ | 满足 $F(m)=0.5$ 的唯一值（$F$ 为总体 CDF） | 真实数据未知；Exp(1) 模拟下 $m=\ln 2\approx 0.6931$ |
| 样本中位数 $\hat{m}$ | 第 $n/2$ 与 $n/2+1$ 个顺序统计量的均值（$n$ 偶） | aircondit7：$\hat{m}=41.5$ h |
| 估计目标 | 用 $\hat{m}$ 推断 $m$，并给出 95% 置信区间 | $\hat{m}\pm$ 区间 |

### 3.2 数据（硬编码 · 可复现）

**真实数据**（Proschan, 1963, *Technometrics* 5(3):375–383；R `boot` 包 `aircondit7`，第 7 架机，n=24，单位：服务小时）：

$$x = (3,5,5,13,14,15,22,22,23,30,36,39,44,46,50,72,79,88,97,102,139,188,197,210)$$

核验后摘要（numpy, `ddof=1`，与 R `sd()` 一致）：

| 统计量 | 值 | 解读 |
|---|---|---|
| $n$ | 24 | 样本量 |
| 中位数 $\hat{m}$ | 41.5 h | 一半故障早于此 |
| 均值 $\bar{x}$ | 64.12 h | $\bar{x}>\hat{m}$ |
| 样本标准差 $s$ | 62.65 h | 离散度大 |
| 极差 | $[3,\ 210]$ | 无异常极小值 |
| $\bar{x}/\hat{m}$ | 1.55 | **明显右偏** → 中位数更稳健 |

> **小样本是优点，不是缺陷（为何 n=24 恰到好处）**：
> - **这正是 Bootstrap 的用武之地**。大样本下中位数近乎已知（如某歌曲库 n=7246 时不确定度仅 ±2 单位），"估计问题"反而消失；n=24 时不确定性真实存在，Bootstrap 才有意义。**数据越少、越偏离正态，越需要 Bootstrap**——直击任务书"Bootstrap 适合该问题的原因"评分点。
> - **t 区间在此不可靠**：小样本 + 强右偏 + 非正态，$\bar{x}\pm t\,s/\sqrt{n}$ 的正态假设失效；Bootstrap 非参数、不假设分布族，对中位数这种无闭式区间的统计量尤其合适。
> - **不能靠合并凑样本**：`aircondit`(n=12) 与 `aircondit7`(n=24) 是两架不同飞机、失效率不同——Proschan (1963) 的核心论点正是"各架机失效率有差异"，合并违反 i.i.d. 且制造假的下降失效率。故 n=24 是该架机的合法全部数据。
> - **数据少 ≠ 计算少**：B=10000 重采样 × R=1000 覆盖实验 ≈ 上百万次中位数计算，分析分量来自重采样实验而非原始数据量。
> - **教科书先例**：Efron & Tibshirani (1993) 经典示例（法学院 n=15、小鼠 n=16、空调 n=12/24）全为小样本；Davison & Hinkley (1997) 更以 `aircondit`（n=12）为全书开篇 Example 1.1——两本 Bootstrap 经典教科书与 `boot` 包均刻意挑小数据配 Bootstrap。

**对照模拟**：$X_i\sim\text{Exp}(1)$，真中位数 $m=\ln 2\approx 0.6931$ 已知，用来证明「我们的 Bootstrap CI 在已知真值下确实覆盖 $\approx 95\%$」，再把同一套方法放心用到真实数据上。

### 3.3 假设

- $x_1,\dots,x_n$ 独立同分布来自某未知分布 $F$（非参数，不假设参数族）。
- $F$ 在 $m$ 处连续且密度 $f(m)>0$（保证中位数唯一、渐近正态）。

### 3.4 「为何用 Bootstrap」——评分核心论证

中位数是任务书四个候选（均值 / 中位数 / 比例 / 两样本均值差）中**最该用 Bootstrap** 的。理由是它的**渐近标准误依赖未知密度** $f(m)$：

$$\sqrt{n}\,(\hat{m}-m)\xrightarrow{d}N\!\left(0,\ \frac{1}{4f(m)^2}\right),\qquad \text{SE}(\hat{m})\approx\frac{1}{2f(m)\sqrt{n}}$$

而对照统计量都有**不依赖未知量**的闭式区间：

| 统计量 | 闭式 95% CI | 是否需要估计未知量 |
|---|---|---|
| 均值 | $\bar{x}\pm t_{\alpha/2,n-1}\,s/\sqrt{n}$ | 否 |
| 比例 | Clopper–Pearson / Wilson | 否 |
| 两样本均值差 | Welch $t$（Satterthwaite $\nu$） | 否 |
| **中位数** | 需先估 $f(m)$（核密度），实操几乎无人采用 | **是** |

> **学术诚实补注**（防被考官追问）：中位数**并非完全没有**非 Bootstrap 区间——顺序统计量 / 符号检验的精确区间 $(X_{(k)},X_{(n-k+1)})$ 不需要 $f(m)$，但只能取离散覆盖水平（给定 $n$ 永远凑不齐 95%）、保守、给不出光滑 SE。Bootstrap 能在任意名义覆盖率、任意统计量下给光滑区间——这才是「中位数是 Bootstrap 最佳教学载体」的完整论证。

## 四、计算实现层（评分：Bootstrap 实现 4 分 + 区间估计 3 分）

### 4.1 非参数 Bootstrap 重采样

对 $b=1,\dots,B$：

1. 从原始样本 $x$ **有放回**等概率抽 $n$ 个，得 $x^{*b}=(x^{*b}_1,\dots,x^{*b}_n)$。
2. 计算 $\hat{m}^{*b}=\text{median}(x^{*b})$。

得经验分布 $\hat{G}^*(t)=\frac{1}{B}\sum_{b=1}^{B}\mathbf{1}\{\hat{m}^{*b}\le t\}$。

### 4.2 点估计与标准误

$$\hat{m}=\text{median}(x),\qquad \widehat{\text{SE}}^*=\sqrt{\frac{1}{B-1}\sum_{b}\bigl(\hat{m}^{*b}-\bar{m}^{*}\bigr)^2},\quad \bar{m}^{*}=\frac{1}{B}\sum_b\hat{m}^{*b}$$

### 4.3 percentile 主方法（95% CI）

$$\text{CI}_{\text{perc}}=\bigl[\hat{G}^{*-1}(0.025),\ \ \hat{G}^{*-1}(0.975)\bigr]$$

即 $B$ 个 bootstrap 中位数的 2.5% 与 97.5% 经验分位数（Efron & Tibshirani 1993, eq 13.5）。

### 4.4 BCa 对照方法（偏差 + 加速度修正）

**偏差修正**（$\Phi$ 为标准正态 CDF，严格 $<$ 计数，与 `scipy.stats.bootstrap` 一致）：

$$\hat{z}_0=\Phi^{-1}\!\left(\frac{\#\{b:\hat{m}^{*b}<\hat{m}\}}{B}\right)$$

**加速度**（jackknife，$\hat{m}_{(-i)}$ 为去第 $i$ 观测后的样本中位数，$\bar{m}_{(\cdot)}$ 为其均值；Efron 1993 eq 14.15，符号按 Efron 约定）：

$$\hat{a}=\frac{\sum_i\bigl(\bar{m}_{(\cdot)}-\hat{m}_{(-i)}\bigr)^3}{6\Bigl[\sum_i\bigl(\hat{m}_{(-i)}-\bar{m}_{(\cdot)}\bigr)^2\Bigr]^{3/2}}$$

**调整分位数**（$z^{(p)}=\Phi^{-1}(p)$）：

$$\alpha_1=\Phi\!\left(\hat{z}_0+\frac{\hat{z}_0+z^{(0.025)}}{1-\hat{a}(\hat{z}_0+z^{(0.025)})}\right),\quad \alpha_2=\Phi\!\left(\hat{z}_0+\frac{\hat{z}_0+z^{(0.975)}}{1-\hat{a}(\hat{z}_0+z^{(0.975)})}\right)$$

$$\text{CI}_{\text{BCa}}=\bigl[\hat{G}^{*-1}(\alpha_1),\ \ \hat{G}^{*-1}(\alpha_2)\bigr]$$

退化自检：$\hat{a}=0,\hat{z}_0=0$ 时 $\alpha_1=0.025$、$\alpha_2=0.975$，BCa 即 percentile。

> ⚠️ **中位数下 BCa 的关键退化（实测确认）**：中位数是**非光滑**泛函，且对偶数 $n$，留一法（jackknife）中位数只取 $\sim 2$ 个值 $\{x_{(n/2)},x_{(n/2+1)}\}$，于是加速度分子 $\sum_i(\bar m_{(\cdot)}-\hat m_{(-i)})^3\approx 0$ → $\hat a\approx 0$。故**中位数场景下 BCa 退化为「percentile + $z_0$ 偏差修正」**，其招牌的二阶（加速度/偏态）修正基本缺席；percentile 与 BCa 在**所有偶数 $n$** 都近乎重合——本项目 Exp(1) 实测（$R=500$）：$n\in\{12,24,48,96\}$ 下 percentile/BCa 覆盖率为 $0.948/0.940,\ 0.944/0.942,\ 0.942/0.936,\ 0.942/0.938$，始终在一倍 MC SE 内。因 $\hat a\equiv0$，BCa 的加速度修正**永不激活**，$n$ 增大也不改变这一点。此为统计量性质、与数据集无关——报告须如实承认这一局限。若想真正展示 BCa 的二阶优势，可另对**光滑统计量**（如 10% 截尾均值）跑同套 Bootstrap 作伴生演示（见 §九 开放问题）。

### 4.5 覆盖率实验（Exp(1) 已知真值 · 方法可信度自证）

```
对 r = 1..R=1000:
    生成 x_r ~ Exp(1), 大小 n
    用同一套 Bootstrap 算法算 95% CI_r
    记 录 I_r = 1{ln2 ∈ CI_r}
覆盖率 p̂ = mean(I_r),  蒙特卡洛 SE = sqrt(p̂(1-p̂)/R)
```

MC SE 一律**按观测 $p̂$ 计算**：$p̂\approx 0.94$ 时 $\approx 0.75$ 个百分点，$p̂\approx 0.95$ 时 $\approx 0.69$ 个百分点（不要套用 $p=0.5$ 的最坏情形上界 $\approx 1.6$ pp，那会高估不确定度约一倍）。

**解析交叉标尺**（核验模拟值是否合理）：Exp(1) 在中位数处 $f(\ln 2)=e^{-\ln 2}=1/2$，故 $\text{SE}\approx 1/\sqrt{n}$；$n=24$ 时 $\text{SE}\approx 0.204$，95% 区间解析全宽 $\approx 2\times 1.96\times 0.204\approx 0.80$。

### 4.7 三因素实验（任务书要求）

1. **样本量 $n$**（在 Exp(1) 模拟上做——真实数据 $n$ 固定为 24 无法扩样；对固定的 24 点经验 CDF 重采样到更大 $n$ 只是在做 bootstrap-of-bootstrap，并不增加信息，演示不了真实的样本量增长）：$n\in\{12,24,48,96\}$，每个 $n$ 从 Exp(1) **抽新鲜样本**、对已知真值 $\ln 2$ 跑 Bootstrap，画 CI 宽 $\sim 1/\sqrt{n}$ 拟合线 + 覆盖率-vs-$n$ → 区间宽近似按 $1/\sqrt{n}$ 收窄、覆盖率随 $n$ 上升逼近名义水平。**并在此 $n$ 扫描上同时报告 percentile vs BCa 覆盖率**：实测 $n\in\{12,24,48,96\}$（$R=500$）下 percentile/BCa 覆盖率始终在一倍 MC SE 内重合（$\hat a\equiv0$，$n$ 增大也不分离）——支撑 §4.4 的退化结论。
2. **重采样次数 $B$**（在 aircondit7 上做）：$B\in\{100,1000,10000\}$，画 CI 上下端点随 $B$ 收敛曲线 → $B\uparrow$ 端点逐步稳定。
3. **数据波动**（在 aircondit7 上做）：向 aircondit7 注入 $k\in\{0,1,2,4\}$ 个极端值（=1000 h），对比「中位数 Bootstrap CI」与「均值 $t$ 区间」宽度变化 → 中位数 CI 几乎不动、均值区间被大幅拉宽，直观证明稳健性。

### 4.8 复杂度

重采样步 $\Theta(nB)$（$B$ 次 × 每次 $O(n)$ 抽样 + $O(n)$ 中位数 via `np.partition`）；BCa 的 jackknife $\Theta(n^2)$（$n$ 次去一中位数），**与 $B$ 无关**——前提是中位数用 $O(n)$ 选择法（`np.partition`）；若改用基于排序的中位数（$O(n\log n)$），jackknife 退化为 $O(n^2\log n)$（这正是 `开题会决议建议.md §二` 核验框的口径，本设计统一取选择法的 $\Theta(n^2)$）。$n=24$ 时增量开销可忽略。percentile 与 BCa 共享同一 $O(nB)$ 重采样，故 BCa 几乎不增加成本。

## 七、结果分析层（评分：结果分析 2 分）

报告里要讲清五点（对应评分点「估计含义、CI 含义、稳定性、局限性+改进、Bootstrap 适合性」）：

1. **估计含义**：$\hat{m}=41.5$ h 表示该架机约一半空调故障发生在起飞后 41.5 小时内；比均值 $\bar{x}=64.12$ h 更代表「典型」故障间隔，因为分布右偏（$\bar{x}/\hat{m}=1.55$）。
2. **CI 含义**：95% percentile CI $[lo,hi]$ 的频率派解释——重复抽样 100 次、约 95 次区间会盖住真中位数 $m$；宽度反映估计的不确定性。
3. **稳定性**：$B=10000$ 下端点波动 $<0.5$ h（图3-3）；$n$ 翻倍 CI 宽约按 $1/\sqrt{2}\approx 0.71$ 收窄（图3-5a，Exp(1) 模拟），与解析 $1/\sqrt{n}$ 标尺吻合。
4. **局限 + 改进**（拿满分的关键）：**实测**（Exp(1)，$n=24$，$R=1000$）percentile 覆盖率 $=0.934\pm0.0079$、BCa $=0.928\pm0.0082$——**轻度欠覆盖**是 percentile 法一阶精度（$O(n^{-1/2})$）在偏态数据下的已知性质，**是发现不是 bug**。**须如实承认**：对中位数（非光滑、偶数 $n$）BCa 的加速度 $\hat a\equiv0$（§4.4，实测精确为 0），故 BCa 在**所有** $n$ 都与 percentile 近乎重合、**并不修正**这一欠覆盖——$n$ 扫描（$R=500$）下两者覆盖率始终在一倍 MC SE 内（$0.948/0.940,\ 0.944/0.942,\ 0.942/0.936,\ 0.942/0.938$）。BCa 的二阶（$O(n^{-1})$）优势要靠**光滑统计量（截尾均值）**才名副其实地显现（§4.4 伴生演示）。覆盖率与 MC SE 一律按观测 $p̂$ 报告、不报裸数字。
5. **为何 Bootstrap 适合中位数**（呼应 3.4）：中位数 SE 依赖未知密度 $f(m)$，闭式区间需密度估计、实操不可行；顺序统计量精确区间虽存在但只能取离散覆盖水平、保守、给不出光滑 SE、不推广。Bootstrap 在任意覆盖率、任意统计量下给光滑区间——这正是中位数场景下「非它不可」的理由。
