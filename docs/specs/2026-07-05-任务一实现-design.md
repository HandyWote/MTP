## 4. 算法要点

- **Jacobi**：`x_new[i] = (b[i] − Σ_{j≠i} A[i,j]·x_old[j]) / A[i,i]`（整轮用旧值）
- **Gauss-Seidel**：同上，但 `x_new` 已算分量立即使用（in-place 更新）
- **停止条件**：`‖x_new − x_old‖∞ < 1e-6` 或达 10000 轮
- **误差历史**：每轮记录 `‖x_new−x_old‖∞`（图4 用）
- **发散检测**：误差连续两轮增大或 `> 1e10` → 提前断（反例用，避免数值爆炸）
- **谱半径**（分析层）：`ρ_J = cos(π/(n+1))`，`ρ_GS = ρ_J²`；GS 快约 2 倍
- **反例**：`A=[[1,2],[2,1]]`，`ρ_J = 2 > 1` → Jacobi 发散
