# 三、任务一：线性代数实验——迭代法解线性方程组

任务目标：理解线性方程组在工程问题中的建模意义，掌握迭代法求解思想，并通过误差曲线和对比图验证算法效果。

## 1. 基本要求

- 将一个实际或模拟问题表示为线性方程组 Ax = b。
- 说明矩阵A、未知量x、右端项b的含义。
- 至少实现一种迭代法，如Jacobi迭代或Gauss-Seidel迭代。
- 设置初值、误差阈值和最大迭代次数，并记录迭代过程。
- 与直接求解结果或理论结果进行对比，分析误差和收敛情况。

## 2. 可视化说明

| 类型 | 建议图表 |
| --- | --- |
| 结构可视化 | 矩阵结构热力图、稀疏性图、系数矩阵规模与元素分布图。 |
| 过程可视化 | 误差随迭代次数变化曲线、不同迭代方法的收敛过程对比图。 |
| 结果可视化 | 数值解与参考解对比图、不同方法的迭代次数/误差/运行时间对比图。 |

可结合问题特点选择矩阵热力图、系数矩阵结构图、迭代误差曲线、残差变化曲线、不同方法结果对比图等可视化方式，用于展示矩阵结构、迭代过程和结果特征。

## 3. 结果说明要求

需要解释迭代是否收敛、收敛速度如何、误差是否满足要求，以及矩阵性质对迭代效果的影响。

# 四、任务二：离散数学实验——三控开关的设计与实现

任务目标：利用离散状态、真值表和布尔逻辑描述三控开关系统，完成逻辑设计、程序实现和可视化验证。

## 1. 基本要求

- 设定三个开关S1、S2、S3，每个开关取值为0或1。
- 枚举全部8种输入状态，建立真值表。
- 设计灯的输出状态L，并写出对应的布尔逻辑表达式。
- 用程序实现三控开关逻辑，能够输入开关状态并输出灯的状态。
- 验证任意一个开关状态变化时，系统输出是否符合设计要求。

## 2. 可视化说明

| 类型 | 建议图表 |
| --- | --- |
| 结构可视化 | 8种状态空间图、真值表、逻辑门结构图或布尔表达式结构图。 |
| 过程可视化 | 状态切换路径图、开关变化前后输出响应图、交互式状态演示截图。 |
| 结果可视化 | 真值表颜色矩阵、输出状态分布图、正确性验证表。 |

可结合问题特点选择状态转移图、真值表、逻辑电路图、开关状态与灯状态示意图、程序输出结果截图等可视化方式，用于展示逻辑结构和验证结果。

## 3. 结果说明要求

需要说明三控开关逻辑如何由状态空间和真值表得到，布尔表达式是否简洁，程序输出是否与真值表一致。

# 五、任务三：概率论与数理统计实验——采用Bootstrap方法解决估计问题

任务目标：理解Bootstrap重采样思想，能够针对均值、中位数、比例、差值等统计量进行估计，并用可视化方式展示不确定性。

## 1. 基本要求

- 选择一个估计问题，明确样本数据、统计量和估计目标。
- 从原始样本中进行有放回重采样，重复多次生成Bootstrap样本。
- 计算每次重采样对应的统计量，形成Bootstrap估计分布。
- 计算点估计结果和置信区间。
- 分析样本量、重采样次数或数据波动对估计结果的影响。

## 2. 可视化说明

| 类型 | 建议图表 |
| --- | --- |
| 结构可视化 | 原始样本分布图、样本箱线图、数据采样结构示意图。 |
| 过程可视化 | Bootstrap估计分布演化图、统计量随重采样次数变化曲线。 |
| 结果可视化 | 置信区间带图、原始估计与Bootstrap估计对比图、分布直方图/密度图。 |

可结合问题特点选择样本分布图、Bootstrap重采样统计量分布图、置信区间示意图、不同参数设置下的结果对比图等可视化方式，用于展示估计分布和结果稳定性。

## 3. 结果说明要求

需要解释估计结果的含义、置信区间的意义、估计分布是否稳定，以及Bootstrap方法适合该问题的原因。
